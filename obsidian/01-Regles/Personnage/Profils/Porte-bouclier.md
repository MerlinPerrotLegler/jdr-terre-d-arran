# Profil : Porte-bouclier

## Description
Le porte-bouclier est un titre qui désigne traditionnellement les guerriers nains chargés de défendre les frontières des cités-forteresses. Lourdement armés, ils gardent la ligne de front et ne reculent devant rien.
