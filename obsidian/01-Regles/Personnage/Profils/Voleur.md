# Profil : Voleur

## Description
La voleuse se faufile dans les ombres ou dans les foules pour s'approprier les biens d'autrui, que ce soit pour son propre compte ou celui d'une tierce personne. Elle est particulièrement à l'aise dans les cités cosmopolites où elle évolue comme un poisson dans l'eau.
