# Profil : Forgeron

## Description
Plus qu'un art, le travail du métal est un véritable mode de vie pour la forgeronne. Ne se sentant jamais mieux que dans la chaleur étouffante de sa forge ou dans les arènes à éprouver le fruit de son œuvre, elle est respectée partout où la guerre se fait par les armes.
