# Profil : Nécromancien

## Description
Rares sont les arts mystiques aussi répudiés que ceux en rapport avec la mort. C'est pourtant vers cette discipline que s'est tourné le nécromancien, pour des raisons qui n'appartiennent qu'à lui, le condamnant à une vie solitaire de paria.
